## 常见问题

### Q1: 找不到 IP 白名单在哪里？
**A:** 有些公众号可能没有 IP 白名单权限。如果你找不到这个设置，说明：
- 你的公众号类型可能不支持 IP 白名单
- 或者需要升级到认证的服务号

### Q2: 配置后还是报错怎么办？
**A:**
1. 等待 5-10 分钟让配置生效
2. 检查 IP 地址是否正确（必须是：111.206.145.24）
3. 如果还有问题，可能需要联系微信客服

### Q3: 可以不配置 IP 白名单吗？
**A:** 不可以。微信 API 强制要求配置 IP 白名单，否则无法调用任何 API 接口。

### Q4: IP 会变吗？
**A:** 如果你的服务器是云服务器，IP 一般不会变。但建议定期检查 IP 是否变化。
