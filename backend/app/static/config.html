配置微信登录凭证

登录 mp.weixin.qq.com 后，URL 中的 token= 后面的数字

按 F12 打开开发者工具 → Network 标签 → 刷新页面 → 复制任意请求的 Cookie

详细说明

第一步：登录微信公众号后台

访问 https://mp.weixin.qq.com 并用微信扫码登录

第二步：获取 Token

登录成功后，浏览器地址栏的 URL 中会有 token=数字，复制这个数字

第三步：获取 Cookie

  1. 按 F12 打开浏览器开发者工具
  2. 切换到 Network（网络）标签
  3. 刷新页面（F5）
  4. 点击任意请求
  5. 在右侧 Headers 中找到 Request Headers
  6. 找到 Cookie，复制整个值（很长的字符串）

重要提示

  • 强烈建议使用小号公众号，不要使用主号
  • 凭证有效期约 4 小时，过期需要重新获取
  • 请勿泄露您的凭证信息
