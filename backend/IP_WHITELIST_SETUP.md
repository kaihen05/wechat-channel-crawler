# 微信公众号 API 配置说明

## 问题说明

尝试通过微信公众号 API 获取文章阅读量时，遇到以下错误：

```
errcode: 40164
errmsg: invalid ip 111.206.145.24 ipv6 ::ffff:111.206.145.24, not in whitelist
```

## 原因

微信公众号 API 有 **IP 白名单** 限制，只有配置在白名单中的服务器 IP 才能调用 API。

## 解决方案

### 步骤 1：登录微信公众平台

使用你的微信公众号账号登录：https://mp.weixin.qq.com

### 步骤 2：进入开发者中心

1. 点击左侧菜单的 **"设置与开发"**
2. 选择 **"基本配置"**

### 步骤 3：配置 IP 白名单

1. 在基本配置页面中，找到 **"服务器配置"** 或 **"IP 白名单"** 部分
2. 点击 **"查看"** 或 **"修改"**
3. 将以下 IP 地址添加到白名单中：

```
111.206.145.24
```

### 步骤 4：保存配置

1. 点击 **"保存"** 或 **"确定"**
2. 配置可能需要几分钟生效

### 步骤 5：重新测试

配置生效后，再次尝试更新文章阅读量。

## 当前服务器信息

- **公网 IP**: 111.206.145.24
- **服务器地址**: 你的当前服务器（可能是腾讯云、阿里云等）

## 注意事项

1. 如果你的服务器 IP 变化，需要及时更新白名单
2. 建议将常用服务器的 IP 都添加到白名单中
3. IP 白名单配置最多可以添加 10 个 IP 地址

## API 限制

微信公众号 API 有以下限制：

- **每分钟调用次数**: 100 次
- **每天调用次数**: 1000 次
- **阅读统计数据**: 只能获取最近 7 天的数据


## 后续操作

配置成功后，可以通过以下方式更新文章阅读量：

1. 在网页管理界面点击 **"从 API 同步阅读量"** 按钮
2. 或调用 API 端点：`POST /api/articles/sync-read-nums-from-api`

参数：
- `begin_date`: 开始日期 (格式: YYYYMMDD)
- `end_date`: 结束日期 (格式: YYYYMMDD)

示例：
```
POST /api/articles/sync-read-nums-from-api?begin_date=20260401&end_date=20260415
```
