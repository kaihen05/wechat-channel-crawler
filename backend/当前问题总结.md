## 常见问题

### Q1: 找不到 IP 白名单设置？
**A:** 可能的原因：
- 你的公众号类型不支持 IP 白名单（需要认证的服务号）
- 或者你使用的是订阅号而非服务号

**解决方案：**
- 确认你的公众号是否是"已认证的服务号"
- 如果不是，可能需要升级公众号类型

### Q2: 配置后还是报错 40164？
**A:**
1. 等待更长时间（5-10 分钟）
2. 检查 IP 地址是否完全正确：`111.206.145.24`
3. 尝试重新添加 IP 白名单
4. 确认你是用管理员账号登录的

### Q3: 可以不配置 IP 白名单吗？
**A:** 不可以。微信 API 强制要求配置 IP 白名单，这是微信的安全机制。

### Q4: 我没有配置权限怎么办？
**A:**
- 检查公众号类型是否支持 IP 白名单
- 联系微信客服咨询
- 或者考虑升级公众号类型
