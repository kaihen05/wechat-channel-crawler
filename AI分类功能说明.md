# AI分类功能说明

## 功能概述

现在支持以下三种方式管理公众号分类：

### 1. 手动选择分类（原有功能）
- 从预设列表中选择：游戏、产品、人力、技术、综合
- 通过编辑按钮修改

### 2. 自定义分类（新功能）
- 输入任意分类名称
- 不再限制为固定的5个分类
- 适合特殊需求的公众号

### 3. AI自动分类（新功能）⭐
- 根据公众号名称智能推荐分类
- 基于关键词匹配 + AI深度分析
- 可手动确认或修改

## 使用方法

### 方法1：添加公众号时AI自动分类

1. 在"我的公众号"页面，使用搜索功能找到公众号
2. 点击"添加"按钮
3. 系统自动调用AI分类
4. 显示推荐的分类，可手动修改

**示例**：
```
搜索：清华学生会
添加 → AI分类：综合
搜索：游戏技术社区
添加 → AI分类：游戏
```

### 方法2：编辑已有公众号的分类

1. 在"我的公众号"列表中，找到要编辑的公众号
2. 点击"编辑"按钮
3. 打开编辑对话框：
   - 查看当前分类
   - 点击"AI分类"按钮获取推荐
   - 或从下拉列表选择
   - 或点击"自定义"输入新分类
4. 点击"保存"

**编辑界面**：
```
┌─────────────────────────────┐
│ 编辑公众号                [×] │
├─────────────────────────────┤
│ 公众号名称：清华学生会      │
│ 大学名称：清华大学         │
│ 分类：[综合 ▼] [自定义] [AI分类] │
│ 或输入自定义分类：_____________ │
├─────────────────────────────┤
│        [取消]  [保存]     │
└─────────────────────────────┘
```

### 方法3：Excel导入时指定分类

Excel 文件中 `category` 列可以是任意值：

| fakeid | nickname | category | university_name |
|--------|----------|-----------|-----------------|
| xxx     | 清华学生会 | 综合      | 清华大学       |
| xxx     | 游戏开发部 | 游戏      | 清华大学       |
| xxx     | 创新创业社 | 创业      | 清华大学       |

## AI分类原理

### 关键词匹配（优先）
系统内置关键词库，快速匹配：

| 分类 | 关键词 |
|------|---------|
| 游戏 | 游戏、电竞、游戏开发、游戏设计、游戏运营、游戏美术 |
| 产品 | 产品、产品经理、产品设计、产品运营、用户体验、产品思维 |
| 人力 | 人力、人力资源、HR、招聘、人才、校园招聘、就业 |
| 技术 | 技术、开发、编程、算法、AI、人工智能、大数据、云计算、前端、后端 |
| 综合 | 学生会、研会、研究生会、校园、学生会、院学生会、校学生会、院研会、校研会 |

### AI深度分析（关键词未匹配时）
- 使用 DeepSeek AI 模型
- 根据公众号名称和描述进行智能分析
- 返回最适合的分类

**示例**：
```
输入：清华学生会
输出：综合

输入：AI算法大讲堂
输出：技术

输入：产品经理实战营
输出：产品
```

## 配置AI功能

### 1. 获取DeepSeek API Key

1. 访问 https://platform.deepseek.com/
2. 注册账号
3. 创建API Key
4. 复制API Key

### 2. 配置API Key

编辑 `backend/.env` 文件：

```env
# DeepSeek AI 配置（用于自动分类）
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_BASE_URL=https://api.deepseek.com
```

### 3. 重启服务

```bash
cd backend
python run.py
```

## API接口

### AI分类接口

**URL**: POST `/api/channels/classify`

**参数**:
- `nickname`: 公众号名称（必填）
- `description`: 公众号描述（可选）

**返回**:
```json
{
  "nickname": "清华学生会",
  "category": "综合",
  "method": "AI"  // 或 "关键词"
}
```

### 更新公众号分类接口

**URL**: PUT `/api/channels/{id}`

**请求体**:
```json
{
  "university_name": "清华大学",
  "category": "游戏"  // 可以是任意字符串
}
```

## 注意事项

### 1. AI分类配置
- 如果未配置 DeepSeek API Key，AI分类将使用关键词匹配
- 配置后可获得更精准的分类建议

### 2. 自定义分类
- 分类字段不再限制为固定值
- 可以输入任意文字作为分类
- 建议使用简短、明确的分类名称

### 3. 分类显示
- 所有自定义分类都会在文章列表中显示
- 首页的固定分类卡片（游戏、产品、人力、技术、综合）仍然存在
- 自定义分类的公众号需要在"全部文章"或搜索中查看

### 4. 数据兼容
- 旧的公众号（未设置分类）会显示"未分类"
- 可以通过编辑功能手动补充分类

## 常见问题

### Q1: AI分类不准确怎么办？
A: 可以手动修改分类，点击"自定义"输入准确的分类名称。

### Q2: 不想配置AI怎么办？
A: 不配置 DeepSeek API Key 时，系统会使用关键词匹配，仍然可以自动分类。

### Q3: 自定义的分类在哪里显示？
A: 自定义分类在文章列表中显示，首页的固定分类卡片不会变化。

### Q4: 能否批量修改分类？
A: 可以通过Excel导入，在category列填写新的分类名称，系统会自动更新。
