# 快速使用指南

## 🚀 前提条件

1. **重启服务器**（重要！）
   ```bash
   # 1. 停止当前运行的服务器
   # 在运行服务器的窗口按 Ctrl+C

   # 2. 重新启动
   cd C:\Users\kaiboy\Desktop\wechat-channel-crawler
   start.bat
   ```

2. **配置微信凭证**（如果还没配置）
   - 访问：http://localhost:8000/config.html
   - 填写微信 token 和 cookie
   - 保存配置

---

## 📋 功能1：公众号阅读量同步

### 方式1：同步时自动更新

1. 访问：http://localhost:8000/manage.html
2. 点击 **"开始同步"** 按钮
3. 等待同步完成
4. 系统会自动更新所有公众号的累计阅读量

### 方式2：手动更新累计阅读量

1. 访问：http://localhost:8000/manage.html
2. 找到 **"📊 更新累计阅读量"** 部分
3. 点击 **"更新累计阅读量"** 按钮
4. 等待更新完成

### 查看效果

1. 访问：http://localhost:8000/
2. 查看文章列表
3. 应该能看到绿色的 **"公众号阅读: XXX"** 标签

---

## ➕ 功能2：新增公众号

### 方式1：通过名称添加（推荐）

1. 访问：http://localhost:8000/manage.html
2. 找到 **"➕ 通过名称添加公众号"** 部分
3. 填写信息：
   - **公众号名称**：例如 "清华大学学生会"
   - **分类**：选择分类（可选）
   - **学校名称**：例如 "清华大学"（可选）
4. 点击 **"添加"** 按钮
5. 系统会自动搜索并添加公众号

### 方式2：通过搜索添加

1. 访问：http://localhost:8000/manage.html
2. 找到 **"🔍 搜索公众号"** 部分
3. 输入关键词
4. 点击 **"搜索"** 按钮
5. 从搜索结果中选择并添加

### 方式3：Excel批量导入

1. 准备Excel文件，包含以下列：
   ```
   nickname          | category | university_name | fakeid（可选）
   -------------------------------------------------
   清华大学学生会     | 综合     | 清华大学       |
   北京大学团委       | 综合     | 北京大学       |
   ```

2. 访问：http://localhost:8000/manage.html
3. 找到 **"📥 批量导入"** 部分
4. 上传Excel文件
5. 查看导入结果

## 🔧 API调用示例

### 更新累计阅读量

```bash
curl -X POST http://localhost:8000/api/articles/update-read-nums
```

### 通过名称添加公众号

```bash
curl -X POST "http://localhost:8000/api/channels/add-by-search?keyword=清华大学学生会&category=综合&university_name=清华大学"
```

### 批量添加公众号

```bash
curl -X POST "http://localhost:8000/api/channels/add-by-name" \
  -H "Content-Type: application/json" \
  -d '["清华大学学生会", "北京大学团委", "复旦大学学生会"]'
```

## ⚠️ 常见问题

### Q: 为什么看不到"公众号阅读"标签？

**A: 请按以下步骤检查：**
1. 确认已重启服务器
2. 点击"更新累计阅读量"按钮
3. 刷新浏览器页面（F5）
4. 打开浏览器控制台（F12），查看是否有错误

### Q: 通过名称添加公众号失败？

**A: 请检查：**
1. 是否已配置微信凭证（token 和 cookie）
2. 公众号名称是否正确
3. 网络连接是否正常
4. 查看浏览器控制台的错误信息

### Q: 同步文章后累计阅读量没有更新？

**A: 请尝试：**
1. 点击"更新累计阅读量"按钮手动更新
2. 检查数据库中是否有文章数据
3. 查看服务器日志是否有错误
