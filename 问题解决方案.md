# 两个问题的解决方案

## 问题1：公众号阅读量同步 ✅

### 实现的功能

1. **同步时自动更新累计阅读量**
   - 每次同步文章时，会自动更新该公众号的 `total_read_num` 字段
   - 计算所有文章的阅读量总和
   - 即使文章已存在，也会更新阅读量（如果微信返回的阅读量有变化）

2. **手动更新累计阅读量**
   - 添加了新的API端点：`POST /api/articles/update-read-nums`
   - 可以手动触发更新所有公众号的累计阅读量
   - 前端添加了"更新累计阅读量"按钮

### 使用方法

#### 方法1：自动同步时更新
```
点击"开始同步"按钮 → 系统自动更新累计阅读量
```

#### 方法2：手动更新
```
访问：http://localhost:8000/manage.html
点击"更新累计阅读量"按钮
```

### API说明

**更新累计阅读量**
```
POST /api/articles/update-read-nums
```

响应示例：
```json
{
  "success": true,
  "message": "成功更新 5 个公众号的累计阅读量",
  "updated_count": 5,
  "total_channels": 8
}
```

## 问题2：新增公众号的方式 ✅

### 实现的功能

1. **通过公众号名称添加**
   - 无需手动填写 fakeid
   - 系统自动搜索公众号并获取 fakeid
   - 支持单个或批量添加

2. **Excel导入（改进）**
   - 不再强制要求 fakeid 列
   - 如果没有 fakeid，会自动搜索获取
   - 只需要提供 nickname 即可

### 使用方法

#### 方法1：通过名称单个添加
```
访问：http://localhost:8000/manage.html
找到"通过名称添加公众号"部分
1. 输入公众号名称（例如：清华大学学生会）
2. 选择分类（可选）
3. 输入学校名称（可选）
4. 点击"添加"
```

#### 方法2：Excel批量导入
```
Excel模板格式：
nickname | category | university_name | fakeid（可选）
---------------------------------------------------
清华大学学生会 | 综合 | 清华大学 | （可选，不填会自动获取）
北京大学团委 | 综合 | 北京大学 | （可选）

上传Excel文件，系统会：
1. 如果有fakeid，直接使用
2. 如果没有fakeid，自动搜索获取
3. 自动进行AI分类
4. 自动识别学校
```

### API说明

**通过搜索添加单个公众号**
```
POST /api/channels/add-by-search
参数:
- keyword: 公众号名称
- category: 分类（可选，默认"未分类"）
- university_name: 学校名称（可选，默认""）
```

**通过名称批量添加**
```
POST /api/channels/add-by-name
参数:
- names: 公众号名称数组
- category: 分类（可选，默认"未分类"）
- university_name: 学校名称（可选，默认""）
```

响应示例：
```json
{
  "success": true,
  "message": "成功添加 3 个公众号，跳过 1 个",
  "stats": {
    "total": 4,
    "success": 3,
    "skipped": 1,
    "errors": 0
  },
  "added_channels": [
    {
      "id": 9,
      "nickname": "清华大学学生会",
      "category": "综合",
      "university_name": "清华大学"
    }
  ]
}
```

## 注意事项

1. **微信凭证配置**
   - 使用"通过名称添加"功能前，需要先配置微信凭证
   - 访问 http://localhost:8000/config.html 配置

2. **AI功能**
   - AI分类和学校识别功能需要配置 DeepSeek API Key
   - 如果未配置，系统会使用关键词匹配的方式

3. **性能考虑**
   - 批量添加公众号时，每个公众号都需要单独搜索
   - 建议一次添加的数量不要太多（建议 < 20个）

4. **累计阅读量更新**
   - 同步文章时会自动更新累计阅读量
   - 如果发现累计阅读量不准确，可以手动点击"更新累计阅读量"按钮
