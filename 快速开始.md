# 公众号累计阅读量功能 - 快速开始

## 📋 问题

文章列表中没有显示"公众号阅读量"绿色标签。

## ✅ 解决方案

**只需重启服务器！**

### 重启步骤

1. 找到运行服务器的命令行窗口（通常显示 `Uvicorn running on http://0.0.0.0:8000`）
2. 按 `Ctrl+C` 停止服务器
3. 重新运行：
   ```bash
   cd C:\Users\kaiboy\Desktop\wechat-channel-crawler
   start.bat
   ```
4. 等待服务器启动完成
5. 访问 http://localhost:8000/

## 🎯 查看效果

重启后，在文章列表中找到标题为 **"测试文章 - 公众号累计阅读量功能"** 的文章，应该能看到：

> 📊 **公众号阅读: 67990** （绿色标签）

## 📊 数据统计

- "测试技术公众号"的累计阅读量：67990
- 其他公众号的累计阅读量：0（因为文章没有阅读量）

只有累计阅读量大于0的公众号才会显示绿色标签。

## 🔧 其他操作

### 更新所有公众号的累计阅读量
```bash
cd backend
python update_channel_reads.py
```

### 添加测试文章
```bash
cd backend
python add_test_article.py
```

## 📝 技术说明

- **后端API**：`/api/articles` 现在会返回 `channel_total_read_num` 字段
- **前端显示**：当 `channel_total_read_num > 0` 时，显示绿色标签
- **数据来源**：公众号所有文章的阅读量总和（`total_read_num` 字段）
