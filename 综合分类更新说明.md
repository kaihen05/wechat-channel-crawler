# 综合分类更新说明

## 更新内容

### ✅ 新增"综合"分类

在原有4个分类（游戏、产品、人力、技术）基础上，新增第5个分类：

- 🌟 **综合** - 专门用于校学生会、院学生会、校研究生会、院研究生会等组织

## 使用方法

### 1. 通过 API 添加"综合"分类公众号
```json
{
  "fakeid": "NjY4NzY1NDMyMQ==",
  "nickname": "清华学生会",
  "category": "综合",
  "university_name": "清华大学",
  "account_name": "qhsu",
  "is_active": true
}
```

### 2. 通过 Excel 导入
Excel 模板中 `category` 列可以选择 "综合"：

| fakeid | nickname | category | account_name | university_name |
|--------|----------|-----------|--------------|-----------------|
| NjY4NzY1NDMyMQ== | 清华学生会 | 综合 | qhsu | 清华大学 |
| MzIwNDU2NzI0NQ== | 北大研究生会 | 综合 | pkuyjsy | 北京大学 |
| MjM5NDk2MzYyMg== | 复旦院学生会 | 综合 | fduystudent | 复旦大学 |

### 3. 前端浏览
访问 http://localhost:8000/，点击"综合"分类卡片，即可查看所有分类为"综合"的公众号文章。

## API 验证

### 获取"综合"分类文章
```bash
curl "http://localhost:8000/api/articles?category=综合&limit=50"
```

### 按综合分类筛选公众号
```bash
curl "http://localhost:8000/api/channels?is_active=true"
# 返回的公众号中 category="综合" 的即为综合分类
```

## 注意事项

1. **分类验证**: API 会严格验证分类值，必须是以下之一：
   - 游戏
   - 产品
   - 人力
   - 技术
   - 综合
   - 未分类

2. **Excel 导入**: 确保 category 列使用正确的分类名称

3. **前端显示**: "综合"分类使用独特的红粉渐变色，便于识别

4. **数据库**: 无需数据库迁移，category 字段已存在
