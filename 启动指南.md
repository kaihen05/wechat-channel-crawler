# 启动指南

## 第一次使用

### 第一步：安装 Python
确保已安装 Python 3.9 或更高版本，下载地址：https://www.python.org/downloads/
安装时勾选 "Add Python to PATH"

### 第二步：安装依赖
打开命令行（CMD 或 PowerShell），执行：
```bash
cd wechat-channel-crawler/backend
pip install -r requirements.txt
```

### 第三步：配置凭证
1. 复制环境变量模板：
```bash
copy .env.example .env
```

2. 用记事本或任意编辑器打开 `.env` 文件，填入以下凭证：

**必填（爬虫功能）：**
- `WECHAT_TOKEN` — 微信公众号后台 token
- `WECHAT_COOKIE` — 微信公众号后台 cookie

**选填（阅读量 API）：**
- `WECHAT_APPID` — 微信公众号 AppID
- `WECHAT_APPSECRET` — 微信公众号 AppSecret

**选填（AI 分类）：**
- `DEEPSEEK_API_KEY` — DeepSeek API Key

> 获取微信凭证的方法：
> 1. 浏览器访问 https://mp.weixin.qq.com 并扫码登录
> 2. 从 URL 中复制 token（一串数字）
> 3. 按 F12 打开开发者工具 → Network → 随便点一个请求 → 复制 Cookie
> 4. 凭证有效期约 4 小时，过期需重新获取

### 第四步：启动服务

**方式一：命令行启动**
```bash
cd wechat-channel-crawler/backend
python run.py
```

**方式二：双击启动**
- `quick_start.bat` — 快速启动（推荐）
- `start.bat` — 带环境检查的启动
- `START_NOW.bat` — 最简启动

### 第五步：访问
- 主页（浏览文章）：http://localhost:8000
- 管理页（添加公众号、同步）：http://localhost:8000/manage.html
- 配置页（设置凭证）：http://localhost:8000/config.html
- API 文档：http://localhost:8000/docs

---

## 日常使用

每次使用前需要更新微信凭证（约 4 小时过期），然后启动服务即可。

数据库 `backend/data/channels.db` 已包含之前收集的所有公众号和文章数据，无需重新收集。

## 项目结构

```
wechat-channel-crawler/
├── backend/
│   ├── app/
│   │   ├── main.py              # FastAPI 主应用
│   │   ├── config.py            # 配置管理
│   │   ├── database.py          # 数据库连接
│   │   ├── models/              # 数据模型
│   │   ├── schemas/             # Pydantic 模式
│   │   ├── routers/             # API 路由
│   │   ├── services/            # 业务服务（爬虫、AI、定时任务）
│   │   └── static/              # 前端页面
│   ├── data/channels.db         # SQLite 数据库（含数据）
│   ├── .env.example             # 环境变量模板
│   ├── requirements.txt         # Python 依赖
│   └── run.py                   # 启动脚本
├── README.md                    # 项目详细文档
├── quick_start.bat              # 快速启动
└── 启动指南.md                   # 本文件
```
